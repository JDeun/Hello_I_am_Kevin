## Current Focus

### OpenClaw

A long-lived personal agent workspace focused on:

- file-native memory
- operational task tracing
- adaptive harnesses
- skill contracts
- durable state capture

### Helm

A reusable open-source operations layer for long-lived agents, built around:

- execution profiles
- context hydration
- manifest-based skill policy
- auditability
- safer extensibility for local and hosted models

## How I Work

- I prefer systems that are explicit rather than magical.
- I reduce operational risk before adding more capability.
- I care about naming, documentation, and structure as much as implementation.
- I try to make tools that still make sense a month later, not just during the first demo.

일하는 방식도 비슷합니다.

- 먼저 구조를 정리하고
- 경계를 명확히 하고
- 나중에 유지보수할 사람 입장에서 설계하고
- 작은 성공을 반복 가능한 시스템으로 바꾸는 쪽을 선호합니다.

---

## Principles

### Simplicity

If a system is hard to explain, it is usually still too vague.

### Clarity

Logs, docs, contracts, and naming should reduce confusion instead of adding ceremony.

### Responsibility

Reliable output requires good boundaries, not just good prompts.

### Durability

A useful tool should survive real use, changing context, and future expansion.
